# Font Resources

## OpenDyslexic Font

The OpenDyslexic font is a free, open-source typeface designed to improve readability for people with dyslexia.

### Installation Instructions

1. Download the font file from:
   - Official: https://opendyslexic.org
   - Alternative: https://www.dafont.com/open-dyslexic.font

2. Place `opendyslexic_regular.ttf` in this directory (`app/src/main/res/font/`)

3. The font will be automatically available in the app as `R.font.opendyslexic_regular`

### Usage

The font is used in dyslexia-friendly theme mode and is referenced in:
- `AccessibleTheme.kt` - Theme definition
- Settings Screen - Toggle dyslexia mode

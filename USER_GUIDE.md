## Getting Started

### System Requirements
- **Minimum Android:** Android 8.0 (API 26)
- **Recommended:** Android 12+ (API 31+)
- **RAM:** 2GB minimum, 4GB recommended
- **Storage:** 50MB free space
- **Internet:** Required for speech recognition

### Installation
1. Download ThriveLearn from Google Play Store
2. Tap \"Install\"
3. Wait for installation to complete
4. Tap \"Open\"

## First Launch

### Welcome Screen
When you first open ThriveLearn, you'll see the **Permissions Screen**.

**Why we need these permissions:**
- 🎤 **Microphone:** For voice dictation and speech-to-text
- 📁 **File Access:** To open documents, audio, and video files

### Granting Permissions

#### Step 1: Read Permission Description
```
Permissions Required

ThriveLearn needs access to your microphone and files to 
provide the best learning experience.
```

#### Step 2: Tap \"Grant Permissions\"
- First permission: **Microphone Access**
  - Tap \"Allow\" when prompted
  - This enables voice dictation

- Second permission: **File Access**
  - Tap \"Allow\" when prompted
  - This enables the document library

#### Step 3: See Confirmation
Once both permissions are granted:
- ✅ Microphone Access: Granted
- ✅ File Access: Granted
- \"All Permissions Granted\" button appears

**You're ready to start using ThriveLearn!**

## Creating Notes

### The Notes Screen

The Notes tab (first tab) is your main workspace.

```
┌─────────────────────────────────┐
│  Quick Notes                    │ ← Title
├─────────────────────────────────┤
│                                 │
│  [Text Input Field]             │ ← Your note content
│                                 │
├─────────────────────────────────┤
│  🎤 Dictate                     │ ← Start voice dictation
│  ✨ Auto-Simplify Text          │ ← Simplify long text
│  💾 Save Note                   │ ← Save permanently
│  🔊 Read Aloud                  │ ← Hear your note
└─────────────────────────────────┘
```

### Method 1: Typing Notes

1. **Tap the text input field**
   - Cursor appears

2. **Type your text**
   - Use device keyboard
   - Text appears in real-time

3. **When done:**
   - Tap **💾 Save Note**
   - Enter a title in the dialog
   - Tap **Save**
   - Note is saved to database

**Example:**
```
Title: Biology Lecture
Content: Photosynthesis is the process by which plants 
         convert light energy into chemical energy...

✅ Saved to database permanently
```

### Method 2: Voice Dictation

#### Starting Dictation
1. **Tap 🎤 Dictate button**
   - Button turns red: \"🎤 Stop Recording\"
   - App says: \"Listening...\"
   - Microphone activates

2. **Speak clearly into microphone**
   - Speak at normal pace
   - Use complete sentences
   - Pause between sentences
   - Text appears as you speak (partial results)

**Dictation Tips:**
- 🎤 Speak 6-12 inches from phone
- 🤐 Minimize background noise
- 📝 Say punctuation: \"period\", \"comma\", \"question mark\"
- ⏸️ Pause between thoughts
- 📢 Speak with confidence (good for low speech clarity)

3. **Tap 🎤 Stop Recording button**
   - Button turns blue: \"🎤 Dictate\"
   - Text is finalized
   - Dictation ends

#### Example Dictation
```
You: \"I learned about photosynthesis today period\"
     \"Plants convert sunlight into energy comma\"
     \"which helps them grow question mark\"

Result in text field:
I learned about photosynthesis today. Plants convert 
sunlight into energy, which helps them grow?

✅ Punctuation automatically added!
```

#### Combining Typing and Dictation
1. Type some text manually
2. Tap Dictate to add more via voice
3. Mix and match as needed
4. Build your complete note

### Saving Your Note

**Option A: Auto-Save**
- If enabled (default): Notes save automatically
- Saves every few seconds while typing
- No action needed from you

**Option B: Manual Save**
1. **Tap 💾 Save Note**
2. **Dialog appears:**
   ```
   Save Note
   
   Enter a title for your note:
   [Text field]
   [Cancel] [Save]
   ```

3. **Enter a title** (or keep default)
   - Example: \"Lesson 5 Summary\"
   - Example: \"Interview Prep\"
   - Example: \"Meeting Notes\"

4. **Tap Save**
   - Toast message: \"Note saved (ID: 42)\"
   - Note is stored permanently in database
   - Text field clears for new note

## Voice Features

### Auto-Simplify Text

**Purpose:** Simplifies long, complex text into key points

**How to Use:**
1. Write or dictate a long note (100+ words)
2. Tap **✨ Auto-Simplify Text**
3. App analyzes text and extracts key sentences
4. Simplified version replaces original
5. Toast confirms: \"Note simplified\"

**Example:**
```
Before (200 words):
Photosynthesis is an incredibly complex biochemical 
process occurring in plants. During this process, which 
happens primarily in the leaves, plants use chlorophyll 
to capture light energy from the sun. This light energy 
is then converted into chemical energy stored in glucose. 
The process involves the light-dependent reactions that 
occur in the thylakoid membranes and the light-independent 
reactions (Calvin cycle) in the stroma...

After (50 words):
Photosynthesis is a complex process in plants. Plants use 
chlorophyll to capture light energy from the sun. This light 
energy is converted into chemical energy in glucose. The process 
involves light-dependent reactions in thylakoid membranes and 
light-independent reactions in the stroma.

✅ Simplified by ~75%!
```

### Read Aloud

**Purpose:** Hear your notes read in natural speech

**How to Use:**
1. Type or dictate text
2. Tap **🔊 Read Aloud**
3. App reads text with text-to-speech
4. Listen to your note
5. Tap again to stop (optional)

**Customizing Playback:**
- Go to **Settings** tab
- Tap **⚙️ Settings**
- Section: **🎤 Voice & Audio**
- Adjust **Speech Rate** (0.5x to 2.0x)
  - Slower (0.8x): Clearer, easier to follow
  - Normal (1.0x): Default speed
  - Faster (1.5x+): For scanning

**Tips:**
- 👂 Listen while multitasking
- 🔊 Great for proofreading
- 📖 Helps memorization
- ♿ Essential for visual impairment

## Document Library

### What Can You Store?

**ThriveLearn supports:**
- 📄 **Text Files:** `.txt` (plain text)
- 🎵 **Audio:** `.mp3`, `.wav`, `.ogg`, `.m4a`, `.flac`
- 🎬 **Video:** `.mp4`, `.mkv`, `.webm`, `.3gp`

**Not supported:**
- ❌ Microsoft Word (`.docx`)
- ❌ PDFs (currently)
- ❌ Images alone
- ❌ Executable files

### Adding Files to Library

**Step 1: Go to Library Tab**
- Bottom navigation → 📚 Library tab

**Step 2: Tap \"Add File to Library\"**
- Button at top of Library screen
- System file picker opens

**Step 3: Navigate and Select**
```
File Picker
├─ Internal Storage
│  ├─ Documents
│  │  ├─ resume.txt ✓ (Supported)
│  │  ├─ guide.pdf ✗ (Not supported)
│  │  └─ notes.docx ✗ (Not supported)
│  ├─ Music
│  │  ├─ lecture.mp3 ✓
│  │  └─ interview.wav ✓
│  └─ Videos
│     ├─ tutorial.mp4 ✓
│     └─ demo.mkv ✓
├─ SD Card (if available)
└─ Cloud Storage (Google Drive, Dropbox, etc.)
```

**Step 4: File Added to Library**
- Toast shows: \"File added to library\"
- File appears in \"Your Study Materials\" list

### Using Library Files

#### Text Files
1. **Tap the file name**
   - Text editor opens
   - File content displayed
   - Large, readable font

2. **Options:**
   - Read with TTS (🔊 Read Aloud)
   - Edit and save changes
   - Tap Close to exit

**Text Editor Features:**
```
┌──────────────────────────────────┐
│ Editing: notes.txt             │

│ [Full text content]              │

│                                  │
├──────────────────────────────────┤
│ [Close] [Save Changes]           │
└──────────────────────────────────┘
```

#### Audio Files
1. **Tap the audio file**
   - Media player opens
   - Shows file name
   - ExoPlayer controls visible

2. **Media Player Controls:**
   ```
   🎵 Lecture Recording.mp3
   ├─ [⏮⏮] Previous (skip back)

   ├─ [─────────●─────] Progress bar
   └─ [00:45 / 45:00] Time display

3. **Tips:**
   - Use skip buttons for chapters
   - Long-press play for slow-motion (0.5x)
   - Volume buttons on device adjust audio

#### Video Files
1. **Tap the video file**
   - Media player opens
   - Video preview shown
   - Full-screen option available

2. **Playback Options:**
   - Play/Pause: ▶ button
   - Skip forward/backward: ▶▶ / ⏮⏮
   - Full screen: Expand icon
   - Adjust speed: In player settings

**Supported Features:**
- 🎬 Resume from last position
- 🔊 Volume control
- ⏩ Playback speed adjustment
- 📺️ Full-screen viewing

## Settings & Customization

### Accessing Settings

1. **Go to Settings Tab**
   - Bottom navigation → ⚙️ Settings

2. **You'll see:**
   ```
   ⚙️ Settings
   ├─ 🎨 Accessibility

### 🎨 Accessibility Settings

#### Display Theme

**Standard Theme:**
- Purple backgrounds
- Dark text
- Material Design 3 colors
- Good for normal vision

**High Contrast Theme (WCAG AAA):**
- Pure black background
- Bright yellow/white text
- Maximum contrast ratio
- Better for low vision
- Better for bright sunlight

**How to Change:**
1. Section: \"🎨 Accessibility\"
2. \"Display Theme\"
3. Tap **Standard** or **High Contrast**
4. Theme changes immediately ✓
5. Auto-saved to device

#### Font Style

**Standard Font:**
- Roboto (Google's default)
- Clean and readable
- Familiar for most users

**Dyslexia Friendly Font (OpenDyslexic):**
- Weighted bottoms on letters
- Increased letter spacing
- Unique character shapes
- Reduces character confusion
- Better for dyslexia

**How to Change:**
1. Section: \"🎨 Accessibility\"
2. \"Font Style\"
3. Tap **Standard** or **Dyslexia Friendly**
4. All text updates immediately
5. Auto-saved

### 🎤 Voice & Audio Settings

#### Speech Rate
**Controls how fast the app reads text aloud**

- Slower (0.5x): Very clear, easy to follow
  - Best for: Clarity, struggling to hear
  - Example: \"Pho-to-syn-the-sis is...\"
  
- Normal (1.0x): Standard speed
  - Best for: General listening
  
- Faster (2.0x): Quick scanning
  - Best for: Review, familiar content
  - Example: \"Photosynthesisisttheprocessby...\"

**How to Adjust:**
1. Section: \"🎤 Voice & Audio\"
2. Slider: \"Speech Rate\"
3. Drag left (slower) or right (faster)
4. Current value shown: \"1.2x\"
5. Changes apply immediately
6. Auto-saved

**Recommended Settings:**
- If hearing issues: 0.7x - 0.9x
- For proofreading: 1.0x - 1.3x
- For review: 1.5x - 2.0x

#### Text Size
**Controls size of text throughout app**

- Smaller (0.8x): More content visible
  - Best for: Scanning, good vision
  
- Normal (1.0x): Default size
  - Best for: Standard reading
  
- Larger (2.0x): Easier to read
  - Best for: Low vision, readability

**How to Adjust:**
1. Section: \"🎤 Voice & Audio\"
2. Slider: \"Text Size\"
3. Drag to desired size
4. Preview shown in real-time
5. Auto-saved

### 💾 Auto-Save Settings

**What is Auto-Save?**
- Automatically saves notes while you type
- Saves every few seconds
- No data loss on crashes
- Prevents accidental deletion

**Toggle Auto-Save:**
1. Section: \"💾 Auto-Save\"
2. Switch: \"Auto-save notes\"
3. Tap to enable/disable
4. Auto-saved

**Benefits When Enabled:**
- ✅ Never lose work
- ✅ Works while offline
- ✅ Saves interrupted thoughts
- ✅ No manual save needed

**When to Disable:**
- ❌ Want manual control only
- ❌ Testing note creation
- ❌ Temporary device use

## Tips & Tricks

### 💡 Productivity Tips

**1. Mix Typing and Voice**
- Type an outline manually
- Use Voice to fill in details
- Provides structured notes

**2. Use Keywords**
- Say topic first: \"Biology lesson\"
- Then add details
- Easy to search later

**3. Save Frequently**
- Use Auto-Save
- Or tap Save every 5 minutes
- Never lose important notes

**4. Organize with Titles**
- Use descriptive titles
- Include date: \"Lesson 5 - June 27\"
- Easy to find later

### 📱 Device Tips

**1. Microphone Placement**
- Hold phone 6-12 inches away
- Point mic toward mouth
- Reduces background noise

**2. Minimize Noise**
- Close other apps
- Lower volume
- Speak clearly
- Good lighting helps focus

**3. Network Issues**
- Speech recognition needs internet
- Use WiFi for best results
- Save notes locally (auto-saved)

**4. Battery Usage**
- Recording uses significant battery
- Consider external speaker
- Enable airplane mode (local features)

### ♿ Accessibility Tips

**For Dyslexia:**
- Enable Dyslexia Friendly font
- Increase text size to 1.5x
- Use high contrast theme
- Reduce speech rate to 0.9x

**For Low Vision:**
- Enable high contrast theme
- Increase text size to 2.0x
- Use larger device (tablet)
- Enable zoom gestures (device settings)

**For Hearing Impairment:**
- Adjust speech rate to slower
- Use captions (future feature)
- Pair with external speaker
- Increase device volume

**For Motor Impairment:**
- Use hardware buttons instead of tapping
- Enable larger touch targets
- Consider stylus for precision
- Use voice commands only

## Troubleshooting

### Dictation Issues

**Problem: \"Dictate\" button does nothing**

Solution:
1. Check microphone permission
   - Settings → Apps → ThriveLearn → Permissions
   - Ensure Microphone is \"Allowed\"
2. Test microphone
   - Open Voice Memos app
   - Try recording
3. Check internet connection
   - Speech recognition requires network
   - Try WiFi instead of mobile data
4. Restart ThriveLearn
   - Close app completely
   - Open again

**Problem: \"Could not understand speech\"**

Solution:
1. Speak more clearly
   - Slow down
   - Enunciate each word
   - Increase volume
2. Reduce background noise
   - Find quiet place
   - Close windows
   - Pause other audio
3. Hold phone closer
   - Microphone should be 3-6 inches from mouth
   - Angle toward mouth
4. Try again
   - Some words are hard for recognition
   - Rephrase and retry

### Notes Not Saving

**Problem: Notes disappear after closing app**

Solution:
1. Enable Auto-Save
   - Settings → 💾 Auto-Save
   - Toggle on
   - Notes save automatically
2. Manually save notes
   - Tap 💾 Save Note button
   - Enter title
   - Tap Save
3. Check storage space
   - Device Settings → Storage
   - Ensure free space available

### Settings Not Persisting

**Problem: Theme resets to default**

Solution:
1. Open Settings tab
2. Select theme again
3. Close and reopen app
4. Theme should persist
5. If not:
   - Clear app cache: Settings → Apps → ThriveLearn → Clear Cache
   - Restart app
   - Reselect theme

### Font Not Changing

**Problem: Dyslexia-friendly font not applying**

Solution:
1. Verify font is installed
   - Font file: opendyslexic_regular.ttf
   - Location: /data/data/com.thrivelearn.app/
2. Restart app
   - Close ThriveLearn
   - Open again
3. Reselect font
   - Settings → 🎨 Accessibility
   - Tap Dyslexia Friendly
4. Reinstall app
   - Uninstall ThriveLearn
   - Reinstall from Play Store

### File Not Opening

**Problem: \"Unsupported file type\" message**

Solution:
1. Check file format
   - ThriveLearn supports: .txt, .mp3, .mp4, .wav, .mkv
   - Not supported: .pdf, .docx, .xlsx
2. Convert file
   - Word to PDF converter
   - Audio file to MP3
   - Video to MP4
3. Try different file
   - Test with known good file
   - Verify file isn't corrupted

### App Crashes

**Problem: ThriveLearn closes unexpectedly**

Solution:
1. Restart app
   - Close completely
   - Wait 10 seconds
   - Open again
2. Clear cache
   - Device Settings → Apps → ThriveLearn
   - Tap \"Storage\" → \"Clear Cache\"
3. Update app
   - Play Store → ThriveLearn → Update
4. Restart device
   - Power off
   - Wait 30 seconds
   - Power on
   - Open ThriveLearn
